A body sensor network (BSN) library that enables the implementation of sensors, data fusers and emergency detection. Refer to \href{https://arxiv.org/pdf/1804.00994.pdf}{\texttt{ https\+://arxiv.\+org/pdf/1804.\+00994.\+pdf}} and \href{https://arxiv.org/pdf/1905.02228.pdf}{\texttt{ https\+://arxiv.\+org/pdf/1905.\+02228.\+pdf}} for more information.

Dependency\+: \href{https://github.com/rdinizcal/lepton}{\texttt{ https\+://github.\+com/rdinizcal/lepton}}

The compilation step-\/by-\/step is taken as follows\+:


\begin{DoxyEnumerate}
\item Clone the repository cointaining \mbox{\hyperlink{namespaceLepton}{Lepton}} and its dependencies
\end{DoxyEnumerate}


\begin{DoxyCode}{0}
\DoxyCodeLine{git clone https://github.com/rdinizcal/libbsn}

\end{DoxyCode}



\begin{DoxyEnumerate}
