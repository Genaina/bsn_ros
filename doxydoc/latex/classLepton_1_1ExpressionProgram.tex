\doxysubsection{Detailed Description}
An \mbox{\hyperlink{classLepton_1_1ExpressionProgram}{Expression\+Program}} is a linear sequence of Operations for evaluating an expression. The evaluation is done with a stack. The arguments to each \mbox{\hyperlink{classLepton_1_1Operation}{Operation}} are first taken off the stack in order, then it is evaluated and the result is pushed back onto the stack. At the end, the stack contains a single value, which is the value of the expression.

An \mbox{\hyperlink{classLepton_1_1ExpressionProgram}{Expression\+Program}} is created by calling create\+Program() on a \mbox{\hyperlink{classLepton_1_1ParsedExpression}{Parsed\+Expression}}.
