\doxysubsection{Detailed Description}
This class is the interface for defining your own function that may be included in expressions. To use it, create a concrete subclass that implements all of the virtual methods for each new function you want to define. Then when you call \mbox{\hyperlink{classLepton_1_1Parser_a735859524488572b39ff012c855c34a0}{Parser\+::parse()}} to parse an expression, pass a map of function names to \mbox{\hyperlink{classLepton_1_1CustomFunction}{Custom\+Function}} objects.
