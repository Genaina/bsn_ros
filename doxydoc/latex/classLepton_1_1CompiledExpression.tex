\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classLepton_1_1CompiledExpression}{Compiled\+Expression}} is a highly optimized representation of an expression for cases when you want to evaluate it many times as quickly as possible. You should treat it as an opaque object; none of the internal representation is visible.

A \mbox{\hyperlink{classLepton_1_1CompiledExpression}{Compiled\+Expression}} is created by calling create\+Compiled\+Expression() on a \mbox{\hyperlink{classLepton_1_1ParsedExpression}{Parsed\+Expression}}.

WARNING\+: \mbox{\hyperlink{classLepton_1_1CompiledExpression}{Compiled\+Expression}} is NOT thread safe. You should never access a \mbox{\hyperlink{classLepton_1_1CompiledExpression}{Compiled\+Expression}} from two threads at the same time.
