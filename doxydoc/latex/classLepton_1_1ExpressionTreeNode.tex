\doxysubsection{Detailed Description}
This class represents a node in the abstract syntax tree representation of an expression. Each node is defined by an \mbox{\hyperlink{classLepton_1_1Operation}{Operation}} and a set of children. When the expression is evaluated, each child is first evaluated in order, then the resulting values are passed as the arguments to the \mbox{\hyperlink{classLepton_1_1Operation}{Operation}}\textquotesingle{}s evaluate() method.
